# Racoon_Gold

Jeu Pygame en Python : Racoon Gold 🦝

1. Installez Python et Pygame.
2. Lancez "racoon_gold.py".
3. Utilisez les flèches pour déplacer le raton laveur.
4. Collectez les pièces, évitez les bombes.
5. Amusez-vous bien ! 😊
